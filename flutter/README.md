## Instalação de dependências

```bash
$ npm install
```

```bash
$ npm install -g @prisma/client
```

```bash
$  npm i -g @nestjs/cli
```

## Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# prisma client
$ npx prisma studio
```
